## 活動内容

屋内外で安定して小さい移動ロボットを自律走行させるためのソフトウェアを研究しています。

- ロボットが様々な環境で安定して自律走行できるようなアルゴリズム
- 屋内と屋外の環境での違いや、障害物の検出・回避、地形の変化に対する適切な制御方法
- 自己位置推定が破綻した際のリセット方法
